2-vazifa

Str string qiymat berilgan, agar satr “f” bilan boshlansa “Fizz” qiymat, Agar “b” bilan tugasa “Buzz” qiymat Console da chiqaring. Agar yuqoridagi har ikki holat ham to`g`ri bo`lsa, “FizzBuzz” qiymatni chiqaring. Qolgan barcha hollarda str o`zgarishsiz qaytsin.



void main(){
 String str='oob';
  if(str.startsWith('f')&&str.endsWith('b')){
    print("FizzBuzz");
  }
 else if(str.startsWith('f')){
    print("Fizz");
  }
  else if(str.endsWith('b')){
    print("Buzz");
  }
   
  else{
    print(str);
  }
}
